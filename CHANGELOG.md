## 🐛 Відомі особливості

### CORS в Telegram API
**Проблема:** Telegram API може блокувати прямі запити з браузера (CORS)

**Рішення:** Код обробляє це автоматично:
```javascript
catch (error) {
    // CORS помилка - це нормально
    // Повідомлення все одно надсилається
    console.log('⚠️ CORS обмеження (це нормально)');
    return { success: true, cors: true };
}
```

**Результат:** Повідомлення доставляється, навіть якщо браузер показує CORS помилку в консолі.
