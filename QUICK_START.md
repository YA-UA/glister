## ⚡ Суперкороткий старт (Telegram)

Якщо хочете почати **зараз** (займе 5 хвилин):

### Крок 1: Створіть бота
1. Відкрийте Telegram → знайдіть **@BotFather**
2. Відправте `/newbot`
3. Введіть назву: `Glister Orders Bot`
4. Введіть username: `glister_orders_bot`
5. **Скопіюйте токен** (довгий код типу `1234567890:ABC...`)

### Крок 2: Отримайте Chat ID
1. Знайдіть вашого бота в Telegram
2. Натисніть START і напишіть `Привіт`
3. Відкрийте в браузері:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
   (Замініть `<ВАШ_ТОКЕН>` на токен з кроку 1)
4. Знайдіть `"chat":{"id": 123456789}` і **скопіюйте число**

### Крок 3: Вставте дані в код
1. Відкрийте файл `js/script.js`
2. Знайдіть на початку файлу:
   ```javascript
   const TELEGRAM_CONFIG = {
       enabled: true,
       botToken: 'YOUR_BOT_TOKEN_HERE',  // 👈 СЮДИ токен
       chatId: 'YOUR_CHAT_ID_HERE'       // 👈 СЮДИ Chat ID
   };
   ```
3. Замініть `YOUR_BOT_TOKEN_HERE` на ваш токен
4. Замініть `YOUR_CHAT_ID_HERE` на ваш Chat ID
5. **Збережіть файл**

### Крок 4: Перевірте!
1. Відкрийте `index.html` в браузері
2. Прокрутіть до форми замовлення
3. Заповніть і натисніть "Оформити замовлення"
4. Перевірте Telegram - має прийти повідомлення! 🎉
