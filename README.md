# Glister - Лендинг зубної пасти

Сучасний одностраничний лендинг для презентації зубної пасти Glister з подвійною інтеграцією Email (EmailJS) та Telegram для отримання замовлень.

## 🚀 ШВИДКИЙ СТАРТ

### Сайт вже працює! Просто відкрийте `index.html` 

### Для отримання замовлень (оберіть варіант):

1. **📱 Telegram (5 хвилин, найлегше)** 👉 `TELEGRAM_SETUP.md`
2. **📧 Email (10 хвилин)** 👉 `EMAILJS_SETUP.md`
3. **🏆 Обидва для надійності** 👉 Дивіться `QUICK_START.md`

**Без налаштувань:** Кнопка Viber вже працює для зв'язку з клієнтами! ✅

### 📱 Telegram Bot інтеграція (НОВЕ!):
При оформленні замовлення через форму:
1. Замовлення **одночасно** відправляється в Email та Telegram
2. Ви миттєво отримуєте **push-уведомлення** на телефон
3. Всі замовлення зберігаються в чаті з ботом
4. Форматоване повідомлення з усіма деталями замовлення
5. Працює паралельно з EmailJS - подвійна надійність!

**Переваги Telegram:**
- ✅ **Безкоштовно** - без обмежень по кількості повідомлень
- ✅ **Миттєво** - push-уведомлення на телефон за секунди
- ✅ **Надійно** - історія всіх замовлень в одному місці
- ✅ **Зручно** - можна працювати в команді (додайте бота в групу)
- ✅ **Без серверу** - працює прямо з браузера

**👥 Робота в команді:**

## 🚀 Як використовувати

### Крок 1: Налаштування EmailJS
1. Прочитайте файл **`EMAILJS_SETUP.md`**
2. Зареєструйтесь на https://www.emailjs.com
3. Налаштуйте email сервіс
4. Створіть шаблон листа
5. Замініть ключі в коді:
   - `index.html` (строка ~14): `YOUR_PUBLIC_KEY`
   - `js/script.js` (строка ~30): `YOUR_SERVICE_ID` та `YOUR_TEMPLATE_ID`

### Крок 2: Налаштування Telegram Bot (НОВЕ!)
1. Прочитайте файл **`TELEGRAM_SETUP.md`**
2. Створіть бота через [@BotFather](https://t.me/botfather) (3 хвилини)
3. Отримайте Bot Token та Chat ID
4. Замініть дані в `js/script.js`:
   ```javascript
   const TELEGRAM_CONFIG = {
       enabled: true,
       botToken: 'YOUR_BOT_TOKEN_HERE',  // 👈 Токен від BotFather
       chatId: 'YOUR_CHAT_ID_HERE'       // 👈 Ваш Chat ID
   };
   ```

### Крок 3: Використання сайту
1. Відкрийте `index.html` у браузері
2. Прокрутіть сторінку для перегляду всіх секцій
3. Заповніть форму замовлення
4. Натисніть "Оформити замовлення"
5. Перевірте ваш email **ТА** Telegram - мають прийти уведомлення!

### Крок 4: Резервний зв'язок
Якщо EmailJS або Telegram не налаштовано або є помилка:
- Клієнт бачить повідомлення з посиланням на Viber
- Плавающа кнопка Viber завжди доступна
- Контакти в футері для прямого зв'язку

### 🎛️ Управління каналами відправки

Ви можете **вимкнути будь-який канал** в `js/script.js`:

```javascript
// Тільки Email (без Telegram)
const TELEGRAM_CONFIG = {
    enabled: false,  // 👈 Вимкнути Telegram
    botToken: 'YOUR_BOT_TOKEN_HERE',
    chatId: 'YOUR_CHAT_ID_HERE'
};

// Тільки Telegram (без Email)
// Закоментуйте виклик emailjs.send() у функції обробки форми

// Обидва канали (рекомендовано!)
const TELEGRAM_CONFIG = {
    enabled: true,   // 👈 Увімкнено
    botToken: '1234567890:ABC...',
    chatId: '123456789'
};
```
