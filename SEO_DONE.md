#### 2. Оновити URL в файлах

Після розміщення замініть `https://yoursite.com` на ваш реальний URL:

**Файли для оновлення:**
- `index.html` (рядки 16, 29)
- `robots.txt` (рядок 11)
- `sitemap.xml` (всі `<loc>` теги)

**Приклад:**
```
Було: https://yoursite.com
Стало: https://username.github.io/glister/
```

**Інструкція:**
📖 Файл: **`SEO_GUIDE.md`** → розділ "Google Search Console"
