## 📝 Оптимізація HTML

### Крок 1: Додати мета-теги в `<head>`

Я зараз додам всі необхідні мета-теги до вашого `index.html`.

#### Що буде додано:

```html
<!-- Базові мета-теги -->
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- SEO мета-теги -->
<title>Glister - Зубна паста без парабенів | Купити в Україні | Доставка НП</title>
<meta name="description" content="Glister - безпечна зубна паста без парабенів, спирту та перекису. Ціна 243 грн. Безкоштовна доставка Новою Поштою по Україні. ☎️ +380 73 725 16 39">
<meta name="keywords" content="glister, зубна паста, паста без парабенів, glister україна, купити glister, зубна паста glister, безпечна паста, натуральна зубна паста">

<!-- Open Graph для соцмереж (Facebook, Viber) -->
<meta property="og:title" content="Glister - Зубна паста для здорових зубів">
<meta property="og:description" content="Безпечна зубна паста без парабенів та спирту. Доставка по Україні.">
<meta property="og:image" content="https://www.genspark.ai/api/files/s/NARKBUHu">
<meta property="og:url" content="https://yoursite.com">
<meta property="og:type" content="website">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Glister - Зубна паста">
<meta name="twitter:description" content="Безпечна зубна паста без парабенів">

<!-- Телефон для мобільних -->
<meta name="format-detection" content="telephone=yes">
<link rel="canonical" href="https://yoursite.com">

<!-- Мова сайту -->
<meta http-equiv="content-language" content="uk">
<meta name="language" content="Ukrainian">

<!-- Для пошукових систем -->
<meta name="robots" content="index, follow">
<meta name="googlebot" content="index, follow">
<meta name="author" content="Glister Ukraine">
```

## 📄 Що таке robots.txt?

**robots.txt** - файл для пошукових роботів, який вказує що можна індексувати.

**Приклад:**
```
User-agent: *
Allow: /
Sitemap: https://yoursite.com/sitemap.xml
```

---

## 🗺️ Що таке sitemap.xml?

**sitemap.xml** - карта сайту для пошукових систем.

**Приклад:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://yoursite.com/</loc>
    <lastmod>2024-03-13</lastmod>
    <priority>1.0</priority>
  </url>
</urlset>
```

## 🔧 Google Search Console

### Як зареєструвати сайт:

1. **Перейдіть:** https://search.google.com/search-console
2. **Увійдіть** через Google акаунт
3. **Додайте сайт:**
   - Натисніть "Додати ресурс"
   - Введіть URL: `https://yoursite.com`
4. **Підтвердіть власність:**
   - Завантажте HTML файл підтвердження
   - АБО додайте мета-тег в `<head>`
5. **Надішліть sitemap:**
   - Розділ "Sitemap"
   - Введіть: `https://yoursite.com/sitemap.xml`
   - Натисніть "Відправити"

**Результат:** Через 1-7 днів сайт з'явиться в пошуку Google.

## 🔧 Яндекс Вебмастер

### Як зареєструвати:

1. **Перейдіть:** https://webmaster.yandex.ru
2. **Увійдіть** через Яндекс акаунт (створіть якщо немає)

5. **Додайте sitemap**

**Результат:** Через 1-7 днів сайт з'явиться в Яндекс пошуку.
