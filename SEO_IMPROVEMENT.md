## 🔍 Чому так відбувається?

### Причина:
Google індексує ваш сайт за **URL** (`ya-ua.github.io`), тому він з'являється за запитом "ya-ua glister".

Запит просто "glister" - **дуже конкурентний**:
- Це міжнародний бренд
- Багато офіційних сайтів
- Потрібен час для просування (2-6 місяців)

**Це нормально!** Всі нові сайти проходять цей етап.

## 🚀 ЩО РОБИТИ ЗАРАЗ (покрокова інструкція)

### Крок 1: Оновити URL в файлах (ВАЖЛИВО!)

Зараз у файлах вказано `https://yoursite.com`, потрібно змінити на ваш реальний URL.

#### 1.1. Оновіть `index.html`

Знайдіть і замініть:

**Рядок ~16** (Open Graph URL):
```html
<!-- Було: -->
<meta property="og:url" content="https://yoursite.com">

<!-- Стало: -->
<meta property="og:url" content="https://ya-ua.github.io/glister/">
```

**Рядок ~30** (Canonical URL):
```html
<!-- Було: -->
<link rel="canonical" href="https://yoursite.com">

<!-- Стало: -->
<link rel="canonical" href="https://ya-ua.github.io/glister/">
```

---

#### 1.2. Оновіть `robots.txt`

**Рядок 11**:
```txt
# Було:
Sitemap: https://yoursite.com/sitemap.xml

# Стало:
Sitemap: https://ya-ua.github.io/glister/sitemap.xml
```

---

#### 1.3. Оновіть `sitemap.xml`

Замініть всі `https://yoursite.com` на `https://ya-ua.github.io/glister/`:

```xml
<!-- Було: -->
<loc>https://yoursite.com/</loc>
<loc>https://yoursite.com/#order</loc>

<!-- Стало: -->
<loc>https://ya-ua.github.io/glister/</loc>
<loc>https://ya-ua.github.io/glister/#order</loc>
```

### Крок 2: Зареєструвати в Google Search Console (ОБОВ'ЯЗКОВО!)

#### 2.1. Перейдіть на Google Search Console
🔗 https://search.google.com/search-console

#### 2.2. Додайте сайт
1. Натисніть **"Додати ресурс"**
2. Виберіть **"Префікс URL"**
3. Введіть: `https://ya-ua.github.io/glister/`
4. Натисніть **"Продовжити"**

#### 2.3. Підтвердіть власність

**Спосіб 1: HTML файл** (найпростіше для GitHub):
1. Завантажте файл підтвердження (буде щось типу `google123abc.html`)
2. Завантажте його в корінь репозиторію GitHub (поряд з `index.html`)
3. Почекайте 2 хвилини (GitHub оновить сайт)
4. Натисніть **"Підтвердити"**

**Спосіб 2: HTML тег**:
1. Скопіюйте мета-тег (виглядає так):
   ```html
   <meta name="google-site-verification" content="abc123xyz...">
   ```
2. Додайте його в `<head>` файлу `index.html` (після інших мета-тегів)
3. Завантажте оновлений `index.html` на GitHub
4. Натисніть **"Підтвердити"**

#### 2.4. Надішліть Sitemap
1. Після підтвердження перейдіть в розділ **"Sitemap"** (ліва панель)
2. Введіть: `sitemap.xml`
3. Натисніть **"Відправити"**

✅ Готово! Google тепер знає про ваш сайт!
