# 👥 Робота в команді через Telegram групу

## 🎯 Що дає додавання бота в групу?

### ✅ Переваги роботи в команді:

#### 1. **Всі бачать всі замовлення**
- 👀 Кожен член команди миттєво бачить нове замовлення
- 📱 Всі отримують push-уведомлення одночасно
- 💬 Можна обговорювати замовлення прямо в чаті

#### 2. **Розподіл роботи**
- ✅ Перший хто побачив - той і обробляє
- 📝 Можна відписатися "Я беру це замовлення"
- 🔄 Легко передати замовлення колезі

#### 3. **Прозорість процесів**
- 📊 Вся історія замовлень в одному місці
- ⏱️ Видно хто і коли відреагував
- 🎯 Легко відслідковувати статуси

#### 4. **Координація**
- 💬 Обговорення проблемних замовлень
- 🤝 Взаємодопомога в команді
- 📞 "Хтось може подзвонити цьому клієнту?"

## 🚀 Як налаштувати роботу в групі

### Крок 1: Створення групи

1. Відкрийте Telegram
2. Натисніть ☰ (меню) → **Нова група**
3. Назва групи: наприклад, "Glister Замовлення"
4. Додайте членів команди
5. Натисніть ✓ (галочка)

---

### Крок 2: Додавання бота в групу

1. Відкрийте створену групу
2. Натисніть на назву групи вгорі
3. Натисніть **Додати учасників**
4. Знайдіть вашого бота (за username, який ви створили)
5. Додайте бота в групу

---

### Крок 3: Надання прав боту

1. У групі натисніть на назву
2. Прокрутіть до вашого бота
3. Натисніть на бота
4. Натисніть ⋯ (три крапки) → **Зробити адміністратором**
5. **Важливо:** Увімкніть дозвіл "Надсилати повідомлення"
6. Натисніть ✓

---

### Крок 4: Отримання Chat ID групи

**Спосіб 1: Через відправку повідомлення**

1. Напишіть будь-яке повідомлення в групі (наприклад, "Тест")
2. Відкрийте в браузері:
   ```
   https://api.telegram.org/bot<ВАШ_БОТ_TOKEN>/getUpdates
   ```
3. Знайдіть розділ з вашою групою
4. Скопіюйте `"chat":{"id": -987654321}`
5. **Важливо:** Chat ID групи завжди **від'ємний** (починається з мінуса)

**Приклад:**
```json
{
  "update_id": 123456789,
  "message": {
    "message_id": 1,
    "chat": {
      "id": -987654321,  // ← ЦЕ ВАШ ГРУПОВИЙ CHAT ID
      "title": "Glister Замовлення",
      "type": "group"
    }
  }
}
```

**Спосіб 2: Через спеціального бота**

1. Додайте в групу бота [@getidsbot](https://t.me/getidsbot)
2. Бот автоматично надішле Chat ID групи
3. Видаліть @getidsbot з групи (вже не потрібен)

---

### Крок 5: Вставка Chat ID в код

Відкрийте `js/script.js` і замініть Chat ID:

```javascript
const TELEGRAM_CONFIG = {
    enabled: true,
    botToken: '1234567890:ABCdef...',
    chatId: '-987654321'  // ← От'ємний номер для групи!
};
```

**⚠️ ВАЖЛИВО:** Chat ID групи завжди від'ємний (з мінусом)!

## 📊 Порівняння: особистий чат vs група

| Функція | Особистий чат | Група |
|---------|---------------|-------|
| Хто бачить замовлення | 👤 Тільки ви | 👥 Вся команда |
| Push-уведомлення | ✅ Тільки вам | ✅ Всім членам |
| Обговорення | ❌ Немає | ✅ Можна обговорювати |
| Розподіл роботи | ❌ Все на вас | ✅ Можна делегувати |
| Контроль якості | ⚠️ Складно | ✅ Всі бачать роботу |
| Історія | ✅ У вас | ✅ У всіх членів |
| Масштабованість | ❌ Тільки 1 людина | ✅ Необмежено членів |

## ⚙️ Налаштування для різних сценаріїв

### Варіант 1: Одна група для всіх

```javascript
const TELEGRAM_CONFIG = {
    enabled: true,
    botToken: '1234567890:ABC...',
    chatId: '-987654321'  // ID групи "Всі замовлення"
};
```

**Плюси:** Всі бачать все  
**Мінуси:** Може бути шумно при великій кількості замовлень

---

### Варіант 2: Декілька груп (потрібно створити кілька ботів)

**Бот 1:** Нові замовлення → Група "Менеджери"  
**Бот 2:** Підтверджені → Група "Склад"  
**Бот 3:** Готові до доставки → Група "Кур'єри"

---

### Варіант 3: Дублювання (особистий чат + група)

Якщо потрібно отримувати копію особисто:

**Рішення:** Створіть 2 конфігурації або відправляйте в 2 місця:

```javascript
// Відправка в групу
await sendToTelegram(orderData, '-987654321');

// Дублювання особисто власнику
await sendToTelegram(orderData, '123456789');
```
