```
┌──────────────────────────────────────────┐
```

## ⚡ Швидке налаштування (3 хвилини)

### 1. Створіть групу
```
Telegram → ☰ Меню → Нова група
Назва: "Glister Замовлення"
Додайте команду
```

### 2. Додайте бота
```
Група → Назва вгорі → Додати учасників
Знайдіть вашого бота → Додати
```

### 3. Зробіть бота адміном
```
Група → Назва → Ваш бот
⋯ (три крапки) → Зробити адміністратором
✓ Надсилати повідомлення → ✓
```

### 4. Отримайте Chat ID групи
```
Напишіть в групі: "Тест"
Браузер → https://api.telegram.org/bot<TOKEN>/getUpdates
Знайдіть: "chat":{"id": -987654321}
```

### 5. Вставте в код
```javascript
// js/script.js
const TELEGRAM_CONFIG = {
    enabled: true,
    botToken: 'ваш_токен',
    chatId: '-987654321'  // ← От'ємне число!
};
```

## 💼 Приклади використання

### Малий бізнес (2-3 людини)
```
Команда:
├── Власник (контроль)
├── Менеджер (приймає замовлення)
└── Кур'єр (доставка)

Результат:
• Власник може відпочивати
• Менеджер приймає замовлення
• Кур'єр бачить що везти
```

### Середній бізнес (5-10 людей)
```
Команда:
├── Менеджери (3 особи)
├── Склад
├── Кур'єри (2 особи)
└── Директор

Результат:
• Менеджери підмінюють один одного
• Склад готує товар відразу
• Кур'єри беруть замовлення самі
• Директор бачить всю роботу
```

### Великий бізнес (10+ людей)
```
Рішення: Кілька груп

Група 1: "Продажі" (менеджери)
Група 2: "Доставка" (кур'єри)
Група 3: "Керівництво" (топ-менеджмент)

Кожна група отримує свої повідомлення
```

## 💬 Як працює в групі

### Приходить замовлення:
```
🎉 НОВЕ ЗАМОВЛЕННЯ #123456

📦 Товар: Glister - 243 грн
👤 Олексій: +380 95 123 45 67

[Всі в групі отримують одночасно]
```

### Команда реагує:
```
Менеджер 1: "Беру, дзвоню клієнту ☎️"
  ↓
Склад: "Готую товар 📦"
  ↓
Кур'єр: "Можу забрати через годину 🚗"
  ↓
Директор: 👍
```

## ⚙️ Корисні налаштування групи

### Закріплене повідомлення
```
📌 ВАЖЛИВО:
• Робочий час: 9:00 - 18:00
• При прийомі пишіть: "Беру ✅"
• Дзвоніть протягом 5 хвилин
```

### Опис групи (правила)
```
📋 ПРАВИЛА:
1. Беру замовлення → пишу "Беру"
2. Дзвоню клієнту → максимум 5 хв
3. Результат → пишу в групу
```

### Швидкі відповіді
```
✅ "Беру"
📞 "Дзвоню"
✍️ "Підтверджено"
📦 "Відправлено"
🚗 "В дорозі"
✅ "Доставлено"
❌ "Відмовився"
```
