# 📱 Налаштування Telegram Bot для отримання замовлень

Ця інструкція допоможе вам налаштувати безкоштовне отримання замовлень у Telegram через Telegram Bot API.

## 📋 Що потрібно

1. Акаунт в Telegram
2. 5 хвилин часу для налаштування
3. Доступ до файлу `js/script.js`

---

## 🤖 Крок 1: Створення Telegram бота

### 1.1. Відкрийте BotFather
1. Відкрийте Telegram
2. Знайдіть [@BotFather](https://t.me/botfather) в пошуку
3. Натисніть **START** для початку діалогу

### 1.2. Створіть нового бота
1. Відправте команду: `/newbot`
2. BotFather запитає назву бота. Введіть, наприклад: `Glister Orders Bot`
3. BotFather запитає username бота (має закінчуватися на `bot`). Введіть, наприклад: `glister_orders_bot`
4. **ВАЖЛИВО:** BotFather надішле вам **токен бота** у форматі:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567
   ```
5. **Збережіть цей токен** - він буде потрібен далі!

### 1.3. Налаштуйте бота (опціонально)
- `/setdescription` - встановити опис бота
- `/setabouttext` - встановити текст "Про бота"
- `/setuserpic` - встановити аватар бота

---

## 💬 Крок 2: Отримання Chat ID

Вам потрібно дізнатися ваш Chat ID, куди бот буде надсилати повідомлення.

### Варіант A: Відправка повідомлення боту (простіший спосіб)

1. Знайдіть вашого бота в Telegram (за username, який ви створили)
2. Натисніть **START**
3. Відправте боту будь-яке повідомлення, наприклад: `Привіт`
4. Відкрийте в браузері:
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
   ```
   **Замініть `<YOUR_BOT_TOKEN>` на токен вашого бота!**
5. Ви побачите JSON відповідь. Знайдіть поле `"chat":{"id": 123456789}`
6. **Збережіть це число** - це ваш Chat ID!

### Варіант B: Використання @userinfobot

1. Знайдіть в Telegram [@userinfobot](https://t.me/userinfobot)
2. Натисніть **START**
3. Бот надішле ваш ID у форматі `Id: 123456789`
4. **Збережіть це число** - це ваш Chat ID!

### Варіант C: Для груп та каналів

Якщо ви хочете отримувати замовлення в групу або канал:

1. Додайте бота в групу/канал як адміністратора
2. Для груп: Chat ID має від'ємний формат (наприклад, `-987654321`)
3. Для каналів: Chat ID має формат `@channel_name` або числовий ID
4. Відправте повідомлення в групу, потім використайте `getUpdates` (Варіант A)

## ⚙️ Крок 3: Налаштування коду

### 3.1. Відкрийте файл `js/script.js`

Знайдіть розділ з налаштуваннями Telegram (біля початку файлу):

```javascript
// ============================================
// НАЛАШТУВАННЯ TELEGRAM BOT
// ============================================
const TELEGRAM_CONFIG = {
    enabled: true,  // Встановіть false для вимкнення Telegram
    botToken: 'YOUR_BOT_TOKEN_HERE',  // 👈 ЗАМІНІТЬ на токен вашого бота
    chatId: 'YOUR_CHAT_ID_HERE'       // 👈 ЗАМІНІТЬ на ваш Chat ID
};
```

### 3.2. Вставте ваші дані

Замініть `YOUR_BOT_TOKEN_HERE` та `YOUR_CHAT_ID_HERE` на ваші реальні дані:

```javascript
const TELEGRAM_CONFIG = {
    enabled: true,
    botToken: '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567',  // Ваш токен
    chatId: '123456789'  // Ваш Chat ID
};
```

### 3.3. Збережіть файл

Збережіть зміни в `js/script.js`.

## 🔧 Додаткові налаштування

### Вимкнення Telegram (залишити тільки Email)

Якщо ви хочете тимчасово вимкнути відправку в Telegram:

```javascript
const TELEGRAM_CONFIG = {
    enabled: false,  // 👈 Змініть на false
    botToken: 'YOUR_BOT_TOKEN_HERE',
    chatId: 'YOUR_CHAT_ID_HERE'
};
```

### Налаштування формату повідомлень

Ви можете змінити формат повідомлень в функції `sendToTelegram()` у файлі `js/script.js`:

```javascript
// Знайдіть змінну message і відредагуйте текст:
const message = `
🎉 НОВЕ ЗАМОВЛЕННЯ #${orderId}

📦 Товар: ${productName}
💰 Ціна: ${productPrice}

👤 Клієнт:
• Ім'я: ${customerName}
• Телефон: ${customerPhone}
• Email: ${customerEmail}

📅 Дата: ${orderDate}
`;
```

### Використання HTML форматування

Telegram підтримує HTML форматування. Змініте `parse_mode` на `HTML`:

```javascript
const payload = {
    chat_id: TELEGRAM_CONFIG.chatId,
    text: message,
    parse_mode: 'HTML'  // Або 'Markdown'
};
```

І використовуйте HTML теги:
```javascript
const message = `
<b>🎉 НОВЕ ЗАМОВЛЕННЯ</b> #${orderId}

<b>📦 Товар:</b> ${productName}
<b>💰 Ціна:</b> ${productPrice}

<b>👤 Клієнт:</b>
• <b>Ім'я:</b> ${customerName}
• <b>Телефон:</b> <code>${customerPhone}</code>
• <b>Email:</b> ${customerEmail}

<i>📅 Дата: ${orderDate}</i>
`;
```

## 🐛 Вирішення проблем

### Проблема: Бот не надсилає повідомлення

**Рішення:**
1. Перевірте, чи правильний токен бота
2. Перевірте, чи правильний Chat ID
3. Переконайтесь, що ви відправили боту хоча б одне повідомлення (натиснули START)
4. Відкрийте консоль браузера (F12) і перевірте помилки

### Проблема: Помилка "Unauthorized"

**Рішення:**
- Токен бота неправильний. Перевірте токен з BotFather

### Проблема: Помилка "Chat not found"

**Рішення:**
- Chat ID неправильний
- Ви не надіслали боту жодного повідомлення (натисніть START)
- Для груп перевірте, чи бот доданий як адміністратор

### Проблема: Повідомлення не приходять, але помилок немає

**Рішення:**
1. Перевірте, чи `enabled: true` в налаштуваннях
2. Відкрийте консоль браузера і перевірте логи
3. Спробуйте відправити тестове повідомлення через браузер:
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/sendMessage?chat_id=<YOUR_CHAT_ID>&text=Test
   ```

### Проблема: CORS помилка

**Рішення:**
- Це нормально! Telegram API може блокувати прямі запити з браузера
- Код автоматично обробляє цю ситуацію
- Повідомлення все одно надсилається (перевірте Telegram)

## 📊 Переваги Telegram Bot

✅ **Безкоштовно** - без обмежень по кількості повідомлень  
✅ **Миттєві повідомлення** - отримуйте замовлення одразу  
✅ **Мобільні сповіщення** - push-повідомлення на телефон  
✅ **Історія замовлень** - всі замовлення зберігаються в чаті  
✅ **Підтримка груп** - працюйте в команді  
✅ **Без серверу** - працює прямо з браузера

## 🔐 Безпека

### Важливо знати:

1. **Не публікуйте токен бота** у відкритому доступі (GitHub, форуми)
2. **Bot токен = повний доступ** до бота
3. **Chat ID не є секретним** - його можна публікувати
4. Токен зберігається в JavaScript коді (видимий у браузері)
5. Для захисту даних клієнтів розгляньте серверну частину

### Рекомендації:

- Регулярно перевіряйте активність бота
- Не надавайте боту зайвих прав в групах
- Якщо токен скомпрометовано - створіть нового бота через BotFather

## 💡 Додаткові можливості

### Кнопки в повідомленнях

Ви можете додати inline кнопки до повідомлень:

```javascript
const payload = {
    chat_id: TELEGRAM_CONFIG.chatId,
    text: message,
    reply_markup: {
        inline_keyboard: [
            [
                {text: "✅ Підтвердити", callback_data: "confirm_" + orderId},
                {text: "❌ Відхилити", callback_data: "reject_" + orderId}
            ],
            [
                {text: "📞 Зателефонувати", url: "tel:" + customerPhone}
            ]
        ]
    }
};
```

**Примітка:** Для обробки кнопок потрібен сервер.

### Фото товару в повідомленні

Надсилайте фото товару разом з описом:

```javascript
// Замість sendMessage використовуйте sendPhoto
const url = `https://api.telegram.org/bot${TELEGRAM_CONFIG.botToken}/sendPhoto`;
const payload = {
    chat_id: TELEGRAM_CONFIG.chatId,
    photo: 'https://your-site.com/product-image.jpg',
    caption: message
};
```
